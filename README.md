# Projeto Power BI: Análise Dinâmica com Parâmetros de Campo

Este projeto demonstra o uso de Parâmetros de Campo no Power BI para criar uma página de análise altamente interativa e limpa

O objetivo foi permitir que o usuário final alterne dinamicamente as métricas principais de todo o relatório (Total Sales, Total Profit e Total COGS) usando uma única seleção de botões

Em vez de criar três páginas separadas ou visuais confusos, foi usada a funcionalidade "Parâmetros de Campo" para criar uma única segmentação de dados

Quando o usuário clica em Total Profit, Total Sales ou Total COGS, o gráfico de cascata muda instantaneamente para refletir essa métrica

Isso melhora drasticamente a Experiência do Usuário (UX), pois mantém a análise focada e reduz a poluição visual
